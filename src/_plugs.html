Виды разъемов

2-pin

Разъем для передачи аудиосигнала для вставных наушников. Обычно на другой стороне кабеля стерео штекер 3,5 мм.

4-pin

Разъем для передачи аудиосигнала для вставных наушников. Обычно на другой стороне кабеля стерео балансный штекер штекер 2,5/4,4 мм.

4.4

Разъем для передачи аудиосигнала для вставных наушников. Обычно на другой стороне кабеля стерео штекер 3.5 мм.

Banana

Разъем для передачи аудиосигнала для вставных наушников. Обычно на другой стороне кабеля стерео штекер 3.5 мм.

BNC

Разъем для передачи аудиосигнала для вставных наушников. Обычно на другой стороне кабеля стерео штекер 3.5 мм.

HDMI

Разъем для передачи аудиосигнала для вставных наушников. Обычно на другой стороне кабеля стерео штекер 3.5 мм.

IEC-C7

Разъем для передачи аудиосигнала для вставных наушников. Обычно на другой стороне кабеля стерео штекер 3.5 мм.

IEC-C13

Разъем для передачи аудиосигнала для вставных наушников. Обычно на другой стороне кабеля стерео штекер 3.5 мм.

IEC-C15

Разъем для передачи аудиосигнала для вставных наушников. Обычно на другой стороне кабеля стерео штекер 3.5 мм.

Lighting

Разъём используемый в портативной технике Apple. Передаёт только цифровые сигналы, вставляется в устройство обеими сторонами.

Mini DisplayPort

Разъём используемый в портативной технике Apple. Передаёт только цифровые сигналы, вставляется в устройство обеими сторонами.

ММСХ

Разъем для передачи аудиосигнала для наушников. Для подключения по балансу к плеерам Sony, Ibasso, Fiio и другим.

RCA

Разъем для передачи аудиосигнала. Используется для соединения аудио-компанентов, таких как усилитель, ЦАП, проигрыватель CD и другим.

RJ-45

Унифицированный разъём, используемый в телекоммуникации. Имеет 8 контактов и фиксатор. Используется для создания ЛВС.

Spade

Spade или «лопатка». Разновидность разъема для подключения к акустическим системам.

Thunderbolt

Spade или «лопатка». Разновидность разъема для подключения к акустическим системам.

Toslink

Cтандартизованная система волоконно-оптических соединителей. Также широко известный как «оптический аудио кабель».

TRS, Jack, 6.3

Разъём для передачи аналогового аудиосигнала. Применяется преимущественно для музыкальных инструментов, микрофонов и клавишных.

USB Type A

Используются в профессиональном и Hi-End оборудовании для повышения устойчивости к помехам и повышения качества звучания.

USB Type B

Используются в профессиональном и Hi-End оборудовании для повышения устойчивости к помехам и повышения качества звучания.

USB Type-C

Спецификация USB для универсального двухстороннего подключения к компьютеру, гаджетам и аудио-устройствам.

Micro-USB

Спецификация USB для универсального двухстороннего подключения к компьютеру, гаджетам и аудио-устройствам.

Mini-USB

Спецификация USB для универсального двухстороннего подключения к компьютеру, гаджетам и аудио-устройствам.

VGA

Спецификация USB для универсального двухстороннего подключения к компьютеру, гаджетам и аудио-устройствам.

XLR

Используются в профессиональном и Hi-End оборудовании для повышения устойчивости к помехам и повышения качества звучания.
